# 설정 탭 기능 명세서

## 1. 목표 설정

### 일일 목표 시간
- 하루 목표 업무 시간 설정 (기본값: 8시간)
- 1~24시간 범위에서 선택
- 타이머 화면에서 목표 대비 진행률 표시에 활용

### 주간 목표 시간
- 주간 목표 업무 시간 설정 (기본값: 40시간)
- 통계 화면에서 목표 달성률 표시에 활용

---

## 2. 알림 설정

### 업무 시작 리마인더
- 특정 시간에 업무 시작 알림 (예: 오전 9시)
- 요일별 설정 가능 (평일만/매일/커스텀)
- ON/OFF 토글

### 휴식 알림
- 연속 업무 시간 후 휴식 알림 (예: 50분마다)
- 25분/50분/90분 선택
- ON/OFF 토글

### 일일 목표 달성 알림
- 일일 목표 시간 달성 시 축하 알림
- ON/OFF 토글

---

## 3. 데이터 관리

### 데이터 내보내기
- CSV 형식으로 업무 기록 내보내기
- 날짜 범위 선택 (전체/최근 1개월/최근 3개월/커스텀)
- 공유 시트를 통해 저장 또는 전송

### 데이터 초기화
- 모든 업무 기록 삭제
- 확인 다이얼로그 2단계 (실수 방지)
- "정말 삭제하시겠습니까?" → "RESET 입력" 확인

- 앱 버전 표시

### 개발자 정보

## 5. 우선순위

### Phase 1 (MVP)
1. 일일 목표 시간 설정
2. 앱 정보 (버전)
3. 데이터 초기화

### Phase 2
4. 주간 목표 시간 설정
5. 휴식 알림
6. 데이터 내보내기

### Phase 3
7. 업무 시작 리마인더
8. 일일 목표 달성 알림
9. 테마 설정 (다크모드)

## 6. UI 구조

```
설정
├── 목표
│   ├── 일일 목표 시간 [8시간 >]
│   └── 주간 목표 시간 [40시간 >]
├── 알림
│   ├── 휴식 알림 [OFF]
│   ├── 업무 시작 리마인더 [OFF]
│   └── 목표 달성 알림 [ON]
├── 데이터
│   ├── 데이터 내보내기
│   └── 데이터 초기화
└── 정보
    ├── 버전 [1.0.0]
    └── 문의하기
```

---

## 7. 데이터 저장

- AsyncStorage 사용 (로컬 저장)
- 키 구조:
  - `@settings/dailyGoal` - 일일 목표 (초 단위)
  - `@settings/weeklyGoal` - 주간 목표 (초 단위)
  - `@settings/breakReminder` - 휴식 알림 설정
  - `@settings/notifications` - 알림 설정 객체
