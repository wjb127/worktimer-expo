# 타이머 탭 UI 개선안

## 현재 구성
- 오늘 날짜
- 상태 표시 (업무 중/대기 중)
- 타이머 (00:00:00)
- 시작/종료 버튼 (원형)
- 오늘 총 업무 시간

---

## 개선안

### 1. 시각적 피드백 강화

#### 타이머 애니메이션
- 업무 중일 때 타이머 주변에 펄스(pulse) 애니메이션
- 시작/종료 버튼 누를 때 스케일 애니메이션
- 숫자 변경 시 부드러운 전환 효과

#### 프로그레스 링
- 목표 시간 대비 진행률을 원형 프로그레스 바로 표시
- 시작/종료 버튼 주변을 프로그레스 링으로 감싸기
- 목표 달성 시 색상 변화 (초록색)

---

### 2. 정보 표시 개선

#### 시작 시간 표시
- 업무 시작 시간 표시 (예: "09:00 시작")
- 예상 종료 시간 표시 (목표 기준)

#### 세션 카운터
- 오늘 몇 번째 세션인지 표시
- 예: "오늘 3번째 업무"

#### 연속 업무일 표시
- 스트릭(streak) 표시
- 예: "5일 연속 업무 중"

---

### 3. 목표 시간 연동

#### 일일 목표 프로그레스
- 목표 8시간 중 현재 진행률 표시
- 프로그레스 바 또는 퍼센트
- 예: "목표 달성률 62%"

#### 남은 시간 표시
- 목표까지 남은 시간
- 예: "목표까지 3시간 2분"

---

### 4. 빠른 액션

#### 메모 추가
- 현재 세션에 간단한 메모 추가
- 무슨 작업 중인지 기록

#### 휴식 타이머
- 별도의 휴식 타이머 버튼
- 포모도로 스타일 (25분 작업 / 5분 휴식)

### 6. 색상/테마

#### 상태별 색상
- 대기 중: 회색 계열 (차분한 느낌)
- 업무 중: 초록색 계열 (활성화)
- 목표 달성: 파란색 또는 골드

#### 다크 모드
- 배경: #1C1C1E
- 텍스트: #FFFFFF
- 액센트: #0A84FF
